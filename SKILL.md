# MCP SymPy

MCP server exposing SymPy symbolic mathematics functionality.

## When to use this skill

Use this skill when you need to:
- Simplify expressions
- Solve equations
- Perform calculus (differentiation, integration)
- Work with matrices
- Number theory computations

## Tools

- **Symbolic**: symbols, sympify, simplify, expand, factor
- **Calculus**: diff, integrate, limit, series
- **Solving**: solve, solveset, nsolve, diophantine, dsolve
- **Matrices**: Matrix operations, inverse, determinant
- **Trigonometry**: sin, cos, tan and inverses
- **Special Functions**: gamma, zeta, erf, fibonacci
- **Number Theory**: isprime, prime, gcd, lcm, totient
- **Sets**: FiniteSet, Interval, Union
- **Logic**: And, Or, Not, Xor
- **Printing**: LaTeX, MathML
